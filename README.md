1. Snake Water Gun
A modern take on the classic 'Rock Paper Scissors.' In this version:

Snake drinks Water (Snake wins)

Water douses Gun (Water wins)

Gun kills Snake (Gun wins)

Features: Randomized computer moves, score tracking, and input validation.

2. The Perfect Guess
A classic number-guessing game where the computer selects a random number, and the player tries to find it.

How it works: After each guess, the program provides feedback (e.g., "Higher" or "Lower") until the user finds the "Perfect Guess."

Features: Tracks the number of attempts and records the user's high score.
